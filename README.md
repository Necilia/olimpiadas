# 🥇 Atletas do Brasil - Buscador Olímpico

Uma ferramenta de busca dinâmica desenvolvida para celebrar e facilitar o acesso a informações sobre os atletas brasileiros. O projeto evoluiu de um layout simples para uma interface moderna, vibrante e totalmente temática.

## 🎨 Design e Interface
O visual do projeto foi totalmente personalizado para trazer a energia das cores nacionais, focando em sofisticação e modernidade:

*   **Fundo Dinâmico:** Implementação de um `linear-gradient` a 135°, garantindo uma transição suave entre as cores da bandeira.
*   **Identidade Visual:** Escolha da fonte **Chakra Petch** para um visual esportivo e tecnológico.
*   **Foco no Conteúdo:** Cards de atletas centralizados e com largura otimizada para uma leitura agradável.

## 🛠️ Desafios Técnicos Superados
Durante o desenvolvimento, algumas soluções de CSS foram fundamentais para a qualidade final:

*   **Fluidez do Layout:** Substituição de alturas fixas por `min-height: 100vh`, permitindo que o fundo acompanhe o crescimento dinâmico da lista de resultados sem quebras visuais.
*   **Efeito de Profundidade:** Uso de `background-attachment: fixed` para manter o fundo estático enquanto o usuário navega pelos resultados.
*   **Lógica de Busca:** Implementação de filtros em JavaScript que ignoram letras maiúsculas/minúsculas para facilitar a pesquisa do usuário.

## 📚 Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica e acessível.
*   **CSS3:** Estilização avançada, uso de Flexbox e gradientes modernos.
*   **JavaScript:** Manipulação dinâmica do DOM e lógica de filtragem de dados.
